The Open Pit Mining Problem

The Problem

Find which combination of blocks in the mine below
will give you the highest profit (but don't forget the rules).

Click Select a Guess, then hover over the mine
and click some blocks to select your answer.

Then Run the algorithm and check your guess!

Rules:

  • Each block can only be excavated once the blocks on top of it have been removed
  • Each block yields a specific profit once it is mined; some are positive and some are negative
